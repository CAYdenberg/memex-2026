## Python notes

**Immutable:** All primitives, strings, tuples, and `bytes`
**Mutable:** All other collections, objects

In python, default parameter values are evaluated at call time, not definition time, which means that if you use a mutable value as a default parameter, calling the function multiple times can have unexpected results.
